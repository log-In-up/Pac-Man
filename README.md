# Pac-Man Unity Project

Прототип 3D аркадной игры в стиле Pacman для Android.

## Начало работы

### Предварительные требования

*   Unity Editor version 2022.3.62f3 or higher.

### Установка

1.  Клонируйте этот репозиторий: `git clone https://github.com/log-In-up/Pac-Man.git`
2.  Откройте этот проект через Unity Hub.
3.  Убедитесь, что все необходимые пакеты установлены через Диспетчер пакетов (`Окно > Диспетчер пакетов`).

## Использование

Откройте файл `MainScene`, расположенный в папке `Assets/Scenes/MainScene.unity`, и нажмите кнопку «Играть».
